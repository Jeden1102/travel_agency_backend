# Travel agency webpage - backend.

Travel agency webpage - backend - work in progress.

# Technologies used

- Strapi

## Installation

1. Build and run dev.

```bash
npm run install
npm run dev
```
2. Copy env.example to .env and insert supabase connection.

```bash
cp .env.example .env
```
